## What I've Built

### 🤖 Nous Meeting — AI-Powered Meeting Intelligence Platform
> Live: [nousmeeting.com](https://nousmeeting.com) · Real-time transcription · OpenAI + AssemblyAI · Celery + Redis async pipelines · Multi-org tenancy · Stripe billing

An AI SaaS platform that turns meetings into structured intelligence. Zoom bot joins calls, transcribes in real time via AssemblyAI, then runs OpenAI-powered summarisation, action item extraction, and speaker-level analysis through a distributed Celery + Redis pipeline. ### 📄 My Text Digest — Document AI SaaS
> Live: [mytextdigest.com](https://mytextdigest.com) · RAG pipeline · OpenAI embeddings · Vector search · Electron desktop + AWS cloud

Full document AI platform shipped in two forms: a local-first Electron + Next.js desktop app with on-device RAG (SQLite vector store), and a cloud SaaS version on AWS (S3 + EC2 workers, PostgreSQL, Stripe). **Stack:** Python · Django · Next.js · Electron · PostgreSQL · AWS S3/EC2 · OpenAI · Celery · Stripe
